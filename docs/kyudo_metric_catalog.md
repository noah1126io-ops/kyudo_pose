# 弓道射型評価用 指標カタログ v0.1

## 目的

本カタログは、弓道動画の射型評価に用いる候補指標を整理し、今後の実装・ラベル設計・教師学習・モバイル移植の優先順位を決めるための基礎資料である。

本カタログでは各指標について、以下を定義する。

- 指標名
- 弓道上の意味
- 関連する八節
- 数値化のしやすさ
- 必要な認識対象
- 客観性の高さ
- 流派差や指導差の大きさ
- ユーザー表示レベル
- MVP 採用可否

## 評価軸

各指標は次の軸で整理する。

### 数値化のしやすさ

- `高`
- `中`
- `低`

### 客観性

- `高`
- `中`
- `低`

### 流派差・指導差

- `小`
- `中`
- `大`

### MVP 採用

- `採用`
- `保留`
- `後段`

### ユーザー表示レベル

- `summary`
  - 総合結果で見せる
- `detail`
  - 詳細画面で見せる
- `internal`
  - 基本は内部利用のみ

## 認識対象の分類

- `人体`
- `矢`
- `弓`
- `本弭/末弭`
- `弦`
- `地面`
- `複合`
